## Part 2: 必备 MCP 服务器（3 个）

> 技能教 Claude **如何做**，MCP 给它**访问外部世界**的能力。

### Tavily
- **功能：** 为 AI 代理构建的搜索引擎，不是蓝色链接 — 干净结构化数据
- **工具：** search、extract、crawl、map
- **安装：** 1 分钟内连接为远程 MCP
- **链接：** https://github.com/tavily-ai/tavily-mcp

### Context7
- **功能：** 将最新库文档注入 LLM 上下文，避免幻觉 API 和过时方法
- **使用：** 在提示词中添加"use context7"
- **支持：** Next.js、React、Supabase、MongoDB 等数千个库
- **链接：** https://github.com/upstash/context7

### Task Master AI
- **功能：** AI 的项目经理，输入 PRD → 结构化任务（含依赖）→ Claude 逐个执行
- **效果：** 将混乱会话转为正规管道
- **兼容：** Cursor、Claude Code、Windsurf
- **链接：** https://github.com/eyaltoledano/claude-task-master
