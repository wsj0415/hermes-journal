## 📐 主动管理基本定律

### 公式

```
IR = IC × √N
```

| 变量 | 含义 |
|------|------|
| **IR** | 信息比率（系统风险调整后收益） |
| **IC** | 信息系数（信号预测与实际市场表现的相关性） |
| **N** | 独立信号数量 |

### 数学对比

**单个强信号**:
```
IC = 0.10
N = 1
IR = 0.10 × √1 = 0.10
```

**50 个弱信号**（每个强度一半）:
```
IC = 0.05
N = 50
IR = 0.05 × √50 = 0.354
```

**结果**: 50 个弱信号系统是单个强信号的 **3.54 倍强大**

## 🎯 5 类机构信号

### 1. 价格动量信号
- **含义**: 定义回溯期内价格运动方向和速率
- **原理**: 短期趋势持续（信息反应不足），中期反转（信息反应过度）
- **公式**: `E(i) = (1/d) × sum of R(i,s)` 最近 d 期
- **关键参数**: d（太短=噪音，太长=滞后）

### 2. 均值回归信号
- **含义**: 资产相对于相关资产的预期公允价值偏离程度
- **原理**: 同行业股票、同事件合约、同标的工具应保持相对定价一致
- **应用**: 关系破裂时激活回归信号

### 3. 波动率信号
- **含义**: 隐含波动率 vs 实际波动率差距
- **原理**: 波动率卖方要求风险补偿
- **交易**: 当隐含>实际时存在差距，收敛时边缘压缩

### 4. 因子信号
- **因子**: 价值、动量、低波动、carry、质量
- **Fama-French 三因子模型**:
```
R(i) = α(i) + β1 × MKT + β2 × SMB + β3 × HML + ε(i)
```
- **每个β**: 敏感性估计，每个因子是独立 IC 的信号

### 5. 微观结构信号
- **内容**: 订单簿深度失衡、买卖价差动态、成交量加权交易攻击性
- **Effective Spread 公式**: `2 × |Trade Price − Mid Price|`
- **含义**: 压缩=低信息不对称，扩张=有人交易未定价信息

## 🔄 11 步阿尔法组合流程

### Phase 1: 数据准备（Steps 1-5）

**Step 1**: 收集每个信号的历史收益序列
```
对于每个信号 i 和每个历史时期 s，记录 R(i,s)
```

**Step 2**: 去除系统漂移（序列去均值）
```
X(i,s) = R(i,s) - mean(R(i,s) across all M periods)
```
将每个信号的收益序列中心化到零，去除趋势扭曲。

**Step 3**: 计算每个信号收益序列的样本方差
```
σ(i)² = (1/M) × sum of X(i,s)²
```
衡量信号波动性，高方差=噪音大，低方差=一致。

**Step 4**: 标准化去均值收益
```
Y(i,s) = X(i,s) / σ(i)
```
所有信号在同一尺度上，无论原始量级如何。

**Step 5**: 仅保留前 M 期，丢弃最新观测
确保权重计算仅使用历史样本外数据，避免过拟合。

---

### Phase 2: 独立性分析（Steps 6-9）

**Step 6**: 横截面去均值
```
Λ(i,s) = Y(i,s) - average of Y(j,s) across all N signals at each period s
```
在每个时间点减去所有信号的平均表现，去除市场-wide 共同驱动因素。

**Step 7**: 仅保留 M-1 期
最终数据卫生步骤，消除任何残留的前瞻信息。

**Step 8**: 计算每个信号的预期 forward 收益
```
E(i) = (1/d) × sum of R(i,s) for most recent d periods
E_normalized(i) = E(i) / σ(i)
```
前向预期估计，缩放到相同单位。

**Step 9**: 计算残差（**关键步骤**）
```
对 E_normalized 对 Λ(i,s) 运行回归（无截距，单位权重）
残差 ε(i) = 独立的前向信息成分
```

**为什么关键**: 不是问哪个信号预期收益最高，而是问哪个信号贡献了最多**独立**的前向信息，不被堆栈中其他信号捕获。

---

### Phase 3: 权重分配（Steps 10-11）

**Step 10**: 设置投资组合权重
```
w(i) = η × ε(i) / σ(i)
```
权重与独立边缘估计成正比，与波动性成反比。
- 高独立边缘 + 低噪音 → 更高权重
- 低独立边缘 + 高噪音 → 更低权重
- **无需主观判断**

**Step 11**: 归一化权重向量
设置 η 使所有权重绝对值之和等于 1。完全分配，无意杠杆。

## ⚠️ 信号相关性破坏边缘

### 隐藏冗余问题

**问题**: 个人信号失败不仅因为噪音，还因为彼此相关，单独检查时无法检测。

**示例**: 动量信号和均值回归信号看似相反，但在某些市场 regime 下，两者可能对同一宏观新闻事件同方向同时反应。

**后果**: 等权重 = 对单一底层原因加倍暴露，而非真正多元化。

### 解决方案

- **Step 6-7 横截面去均值**: 识别并去除共享成分
- **Step 9 回归**: 隔离每个信号的独特贡献
- **结果权重**: 仅分配价值给真正独立的信息

### N 的真正含义

```
IR = IC × √N
```

**N 不是堆栈中信号数量，而是去除共享方差后的有效独立信号数量。**

- 50 个相关信号 ≈ 10-15 个独立信号的多元化收益
- 50 个真正独立信号 = 全部 50 个的收益

**实践后果**: 相信自己运行 20 个独立信号的 PM 可能只运行 6 个有效信号。20 个独立观点合理的仓位规模对 6 个来说太大了。这种杠杆不匹配是大多数系统策略爆仓的机制。

## 💡 零售交易者的系统性错误

**Polymarket 4 亿笔交易研究发现**:

| 市场定价 | 实际 YES 决议率 |
|----------|----------------|
| 5% - 15% | 4% - 9% |

**差距原因**: 交易者运行不完整模型，而非事件真的那么不可能。

**零售普遍问题**:
- 单一模型
- 单一数据源
- 单一概率估计

**机构优势**: 组合引擎将 5+ 信号转换为单一组合估计，数学上比任何单独组件更准确。
