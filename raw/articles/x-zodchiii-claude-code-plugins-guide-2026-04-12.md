## 什么是 Claude Code 插件

> **插件是技能/命令/hooks/MCP 服务器的捆绑包，用一个命令安装。** 它们扩展 Claude Code 能做的事，无需你写任何配置。

**安装方式**（所有插件相同）:
```
1. 在终端打开 Claude Code
2. 输入 /plugin
3. 去 Discover 标签
4. 按名字找到插件
5. 选择并选择安装范围（用户或项目）

一个命令，一次点击。
```

**社区插件**（不在官方市场）:
```bash
/plugin marketplace add owner/repo
```
然后该 repo 的插件会出现在 Discover 标签。
