## 核心理念

### 大多数人如何使用 Claude

```
1. 打开 Claude
2. 输入问题
3. 阅读答案
4. 关闭标签
5. 继续一天，完全 unchanged
```

**这就像计算器** — 节省几分钟，然后继续过日子。

---

### Claude 真正是什么

**不是更好的搜索引擎**，**不是更快的写邮件方式**。

**正确使用**，它更像是拥有：
- 高级分析师
- 治疗师
- 律师
- 程序员
- 战略家
- 研究团队

**全部在一个地方，凌晨 2 点可用，从不疲倦，从不安小时收费**。

## 10 个完整系统

### 1. 生活审计系统（Life Audit）

**覆盖**: 9 领域 + 心理挖掘

**领域**: 职业/金钱/健康/心理/关系/浪漫/环境/目标/成长

**流程**:
```
Phase 1 — 心理挖掘
- 诚实基线（1-10 分，持续不满，重复抱怨）
- 反愿景（5 年/10 年/生命尽头）
- 身份和恐惧映射

Phase 2 — 9 领域审计
- 每领域总结后继续
- 总是推动具体答案

Phase 3 — 完整诊断
- 根模式（2-3 段）
- 逐领域分解
- 真正的敌人
- 什么处于危险中

Phase 4 — 生活架构
- 反愿景（1 句）
- 愿景 MVP（2-3 句）
- 身份声明（3 个"I am the type of person who..."）
- 1 年镜头 | 90 天任务 | 30 天 boss fight
- 每日杠杆（正好 3 个）
- 约束（3 个立即停止的事）
- 每周重置（5-7 个周日问题）

Phase 5 — 实施计划
- 每领域 START/STOP/CHANGE
- 环境设计动作
- 30 天实验

Phase 6 — 每日协议
- 早晨（首 60 分钟）
- 深度工作块
- 午间中断
- 晚间放松
- 每周锚点（周日仪式）
```

**你得到**:
- 根因心理挖掘（非症状追逐）
- 9 领域审计强制真实答案
- 书面诊断：根模式/领域分解/真正敌人/危险什么
- 生活架构：愿景/反愿景/身份/90 天任务/每日系统
- 领域计划：开始/停止/改变/环境设计/30 天实验
- 每日协议：复合的最小可行日

### 2. 商业审计系统（Business Audit）

**覆盖**: 8 领域 + 绑定约束诊断

**领域**: 收入模型/产品/客户/营销/运营/团队/财务/创始人角色

**流程**:
```
Phase 1 — 诚实基线
- 业务做什么，服务谁，典型周什么样
- 当前月收入，6 个月趋势
- 如果业务明天消失谁会真正想念
- 最害怕看什么
- 告诉自己没问题但实际知道不是的

Phase 2 — 8 领域审计
1. 收入和业务模型
2. 产品或服务
3. 客户和市场
4. 营销和获取
5. 运营和交付
6. 团队和人
7. 财务和数字
8. 创始人角色

Phase 3 — 诊断
- 绑定约束（一段，一件事）
- 逐领域分解
- 真正问题（创始人避免看到的信念/行为/决策）
- 危险什么（12 月/3 年/5 年如果不变）

Phase 4 — 商业架构
- 反愿景（3 年如果不变）
- 愿景（2 句，具体）
- 90 天约束（一个焦点，清晰成功指标）
- 30 天构建（具体系统或交付物）
- 每周运营节奏（3 个周一审查指标）
- 3 个消除（立即停止的事）

Phase 5 — 实施计划
- 每领域：本周开始/立即停止/30 天构建系统/流程变更/30 天实验
```

**规则**:
- 从不接受收入数字不问利润率
- 从不接受"我们在增长"不问增长率和驱动什么
- 从不让创始人责怪市场不检查自己角色
- 推动每个模糊答案
- 目标是下季度不同运营的业务

**你得到**:
- 8 领域完整业务审计
- 清晰绑定约束诊断
- 创始人角色直接评估
- 聚焦 90 天计划 + 成功指标
- 3 个立即停止的事
- 基于 3 个真实指标的每周节奏

### 3. 财务审计系统（Money Audit）

**覆盖**: 7 领域 + 财富心理学

**领域**: 收入/支出和生活方式/债务/储蓄和应急基金/投资和财富建设/财务心理学/财富目标

**流程**:
```
Phase 1 — 诚实基线
- 当前月到手收入
- 实际每月支出（非预算，实际）
- 当前净资产
- 诚实思考财务状况的感觉
- 避免直接看的财务真相

Phase 2 — 7 领域审计
1. 收入（所有来源，轨迹，天花板，2x 障碍）
2. 支出和生活方式（最大三项，尴尬支出，未使用订阅）
3. 债务（每笔债务：金额/利率/最低支付，情感关系）
4. 储蓄和应急基金（流动资产，月数，自动或手动）
5. 投资和财富建设（持有，策略，收入百分比，不理解什么）
6. 财务心理学（成长家庭金钱含义，继承信念，运行在哪里）
7. 财富目标（具体数字/生活方式/时间线，当前轨迹，差距）

Phase 3 — 诊断
- 根模式（核心信念或行为）
- 逐领域分解
- 金钱故事（继承信念，来自哪里，成本什么）
- 危险什么（10 年当前轨迹产生什么）

Phase 4 — 财富架构
- 数字（具体净资产目标和时间线）
- 储蓄率（确切百分比，立即开始）
- 债务序列（消除顺序和时间线）
- 投资系统（账户类型/工具/自动贡献）
- 收入动作（下 90 天增加收入的一件事）
- 3 个消除（立即削减的支出类别）
- 每周金钱仪式（20 分钟实践）

Phase 5 — 30 天财务重置
- 周 1：审计和削减
- 周 2：系统（自动化，开户，设置）
- 周 3：收入动作
- 周 4：投资开始（首次贡献/首次持仓/首次自动转账）
```

**规则**:
- 从不接受"我不真的追踪支出" — 帮助从记忆估算
- 从不让情况仅归咎收入不检查储蓄率和心理学
- 推动每个整数 — 整数是估计，估计隐藏真相
- 从不让对话保持舒适
- 金钱回避是大多数人最昂贵习惯

**你得到**:
- 7 领域审计：财务 + 心理学
- 继承金钱故事（及如何运行选择）
- 确切财富架构：储蓄率/债务顺序/投资系统/下收入动作
- 4 周重置计划：审计→系统→收入→投资
- 每周 20 分钟金钱仪式
- 3 个不符合目标的支出削减

### 4. 关系审计系统（Relationship Audit）

**适用**: 浪漫关系/亲密友谊/家庭动态/跨关系模式

**覆盖**: 沟通/需求和欲望/模式/依恋和历史/真正问题

**流程**:
```
Phase 1 — 诚实基线
- 一段话描述关系（好/坏/不常说的）
- 1-10 满意度，具体什么使那数字
- 最想改变但从未直接问的事
- 爱你但完全诚实的人会说什么
- 关于自己角色最难承认真相

Phase 2 — 关系审计
- 沟通（最近冲突，说什么 vs 意味什么，从不直接说什么）
- 需求和欲望（需要什么没得到，如何清晰沟通，对方需要什么没给）
- 模式（重复争吵/张力，完整周期，来自哪里）
- 依恋和历史（成长时成人如何相爱，学到什么，编程运行在哪里）
- 真正问题（5 年不变可接受吗，真正想要什么，害怕承认什么）

Phase 3 — 诊断
- 动态（命名两人间运行模式）
- 他们角色（具体和公平）
- 你的角色（命名不软化）
- 依恋伤口（潜在恐惧或信念，来自哪里）
- 分叉（两条诚实路径：真正修复需要什么，距离/离开看起来像什么）

Phase 4 — 计划
- 一次对话（具体需要说什么，如何开始，目标什么结果）
- 三个改变（用户承诺改变的具体行为）
- 三个请求（直接问的具体事）
- 30 天实验（动态具体改变 30 天，清晰衡量是否工作）
- 决策点（如果 X 在 Y 时间框架内不发生，决策是 Z）
```

**规则**:
- 从不验证不满不问用户贡献
- 从不让对方是整个问题
- 推动任何排练版本故事 — 排练版本从不是真实版本
- 不推荐离开或留下 — 摆出两条路径让用户 full information 决策
- 目标是清晰，非舒适

**你得到**:
- 动态清晰地图（两角色 + 核心伤口）
- 你的贡献，诚实评估
- 确切对话启动（说什么 + 如何开始）
- 3 个行为改变你承诺
- 30 天实验带成功指标
- 两条路径，清晰：修复或距离，所以 full information 决策

### 5. 职业转型审计系统（Career Pivot Audit）

**覆盖**: 从什么跑/向什么跑/真实资产/实际现实/真正恐惧

**流程**:
```
Phase 1 — 诚实基线
- 目前做什么，多久
- 1-10 工作感觉多适合
- 具体什么使想改变（非表面原因，下面真实原因）
- 以前试过改变职业/角色吗，发生什么
- 想象实际适合的工作版本什么样

Phase 2 — 职业审计
- 从什么跑（具体恨什么，什么会跟随到新职业，害怕避免什么）
- 向什么跑（实际想要工作生活看起来像，何时最 alive/engaged，免费做什么）
- 真实资产（真正擅长什么，人们反复求助什么，独特知识/技能/经验）
- 实际现实（财务跑道，非谈判，网络中谁已在目标方向，知识/证书差距）
- 真正恐惧（具体生动失败版本，失去什么身份，别人说什么，需要相信什么）

Phase 3 — 诊断
- 真正问题：职业转型问题还是不同问题穿职业转型服装
- 方向：基于所说 — 什么方向实际适合
- 差距：技能和现在之间具体什么 — 技能/网络/证书/金钱/身份
- 时间线：成功过渡现实估计多久
- 风险：如果转型不工作实际什么处于危险

Phase 4 — 转型计划
- 目的地：目标角色/领域/工作类型清晰陈述
- 桥梁：如何从现在到那里不破坏现有 — 具体序列
- 90 天实验：下 90 天做什么测试方向 — 具体动作可衡量结果
- 第一概念验证：下 30 天尝试新方向最小版本
- 网络动作：下 2 周联系三具体人已在想去地方
- 技能差距：下 90 天学习一件事关闭最重要差距
- 财务桥梁：过渡期间如何维持收入 — 具体选项
```

**规则**:
- 从不验证转型欲望不先检查问题是否可不离开解决
- 从不让用户用职位定义目标 — 推动每日体验和影响
- 从不接受"我对...有激情"作为职业策略 — 推动技能和市场证据
- 目标是适合工作，非新职位

**你得到**:
- 清晰这是职业问题还是其他东西
- 从什么跑 vs 向什么跑
- 可转移优势 no-BS 审计
- 基于适合的具体方向
- 90 天试驾（无焦土）
- 过渡财务桥梁

### 6. 心理清晰度审计系统（Mental Clarity Audit）

**覆盖**: 6 领域心理静态

**领域**: 思想循环/未做决策/未说事/情感积压/身份噪音/环境和输入

**流程**:
```
Phase 1 — 诚实基线
- 当前心理状态诚实描述（非"压力大" — 头里面实际感觉像什么）
- 何时最后感觉真正清晰 — 平静/聚焦/敏锐，什么不同
- 什么想法反复回来没完全处理
- 假装什么没事但实际不是
- 如果 mind 能说话会说什么最需要

Phase 2 — 清晰度审计
1. 思想循环（反复返回什么，每思想实际关于什么，什么未做决策/未说/未承认）
2. 未做决策（列表每个避免决策，为什么没做，成本什么）
3. 未说事（需要对谁说什么没说，需要对自己承认什么，避免对话）
4. 情感积压（携带什么情绪没完全感觉/处理，愤怒什么，grief 什么，害怕什么）
5. 身份噪音（试图成为谁不适合实际是谁，表演什么，谁期望从不是自己）
6. 环境和输入（消费什么增加噪音，物理环境影响，多少时间真正安静）

Phase 3 — 诊断
- 核心杂乱：产生最多心理噪音的两三件事
- 避免模式：mind 通过保持忙碌和模糊保护什么
- 主要循环：保持其他运行的思想模式
- 本周需要发生什么：7 天内发生会缓解最多压力的具体动作/对话/决策

Phase 4 — 清晰度协议
- 大脑转储：完整书面协议清空 mind 到纸上 — 确切写什么/多久/多频繁
- 决策冲刺：单次聚焦会话做每个未做决策方法 — 如何运行
- 对话框架：如何有需要的具体对话 — 说什么/如何开始/目标什么结果
- 每日清晰度实践：最小每日实践 — 非完整系统，实际最小 — 保持清晰度一旦有
- 输入审计：下 30 天确切减少/消除什么信息消费
- 环境改变：今天做一个物理或数字改变
- 每周 mind-clear：周日仪式 — 写什么/决定什么/释放什么 — 防止积压重建
```

**规则**:
- 从不让用户停留表面 — "压力大"不是答案 — 具体什么
- 从不接受"我只需要更自律"作为诊断 — 几乎总是避免
- 跟随每个听起来重要的线索
- 不匆忙挖掘到协议 — 如果情感表面出现，停留
- 清晰度第一，系统第二

**你得到**:
- 6 类别完整 mind-clearing 挖掘
- 主要噪音来源具体诊断
- 单次会话方法清除未做决策
- 需要对话框架
- 最小每日清晰度实践
- 防止复发的每周仪式

### 7. 健身审计系统（Fitness Audit）

**覆盖**: 训练/营养/睡眠/能量/身体形象 — 心理学优先

**流程**:
```
Phase 1 — 诚实基线
- 当前健身状态诚实描述
- 何时最后感觉身体良好
- 什么习惯反复开始和放弃，真实原因
- 身体状态多少是心理和情感状态症状
- 如果身体反映如何对待自己说什么

Phase 2 — 健身审计
- 训练（当前计划，何时最后 energized，什么会改变如果 6 个月无收入压力）
- 营养（实际吃什么，什么情绪驱动饮食，什么会改变如果相信值得）
- 睡眠（实际睡眠多少，质量如何，什么干扰）
- 能量（日常能量水平，什么耗尽什么补充）
- 身体形象（如何感觉关于身体，什么恐惧形状如何出现）

Phase 3 — 诊断
- 根模式：驱动每个问题的核心信念或行为
- 逐领域分解：当前现实/核心障碍/9/10 看起来像/解锁最多一个改变
- 健身故事：继承信念运行健身决策，来自哪里，成本什么
- 危险什么：当前轨迹 10 年实际产生什么

Phase 4 — 健身架构
- 数字：具体目标（体脂/肌肉/表现）和时间线
- 训练系统：具体计划，自动贡献设置
- 营养系统：具体饮食方法，自动餐食准备
- 睡眠系统：具体睡前仪式，自动关闭时间
- 收入动作：下 90 天增加收入一件事
- 3 个消除：立即削减的支出类别
- 每周金钱仪式：20 分钟实践

Phase 5 — 30 天健身重置
- 周 1：审计和削减
- 周 2：系统（自动化，开户，设置）
- 周 3：收入动作
- 周 4：投资开始（首次贡献/首次持仓/首次自动转账）
```

**你得到**:
- 7 领域审计：财务 + 心理学
- 继承金钱故事（及如何运行选择）
- 确切财富架构：储蓄率/债务顺序/投资系统/下收入动作
- 4 周重置计划：审计→系统→收入→投资
- 每周 20 分钟金钱仪式
- 3 个不符合目标的支出削减

## 会话规则（通用）

所有 10 系统共享相同核心规则：

| 规则 | 含义 |
|------|------|
| **从不接受模糊答案** | 总是推动具体 |
| **从不给通用建议** | 每建议基于这人具体所说 |
| **从不表面动机** | 说什么是真实 |
| **总是移动前反映** | 包括什么没说 |
| **推动舒适** | "那听起来像你会给别人的答案，真实的是什么？" |
| **目标不是好对话** | 目标是生活改变 |
