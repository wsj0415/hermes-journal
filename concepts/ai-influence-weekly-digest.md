## 编译知识

AI 影响力周报是一个自动化内容筛选和提炼系统，用于从 65 个 AI 领域影响力账号中捕捉高价值信息。

**核心价值:**
- **信息过滤:** 从 87 条候选推文中精选 8 条 (9% 通过率)
- **趋势识别:** 识别跨账号的共同主题和新兴模式
- **可复用性:** 每条精选内容附带"为什么有用"和实操建议

**工作流程:**
1. 扫描 20 个核心账号的近期推文
2. 收集候选内容 (约 80-100 条/周)
3. AI 筛选和分类 (可复用方法/工作流优化/新工具)
4. 生成 Markdown 报告和 HTML 可视化
5. 存档到知识库 `raw/articles/`

**分类框架:**
- 🛠️ **可复用方法** - 可直接迁移的 prompt 模式和工作流
- 💡 **工作流优化** - 提升 AI 协作效率的实践技巧
- 🚀 **新工具** - 新发布的能力和产品

**第一期核心洞察 (2026-04-17):**
- Agent 上下文管理成为焦点 (Claude Code 1M 窗口)
- Prompt Engineering 从"技巧"转向"流程工程"
- 多模态生成进入实用阶段 (Gemini 3 单 prompt 构建应用)
