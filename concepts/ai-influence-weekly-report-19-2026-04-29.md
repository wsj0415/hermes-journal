# AI影响力周报 第 19 期 — 核心洞察 (2026-04-29)

> 汇总自 [[ai-influence-weekly-report-19-2026-04-29|AI影响力周报 第 19 期]]

## 核心趋势：AI 编程从"辅助"走向"委托"

### 关键洞察

1. **Karpathy 确认范式转变**：2025年12月是 coding agents 的分水岭。之前不可用，之后可以完全委托。"Agentic Engineering" 成为新范式 — 用英语给 agent 分配任务，并行管理。

2. **Spec-Driven Development 方法论**：Andrew Ng 与 JetBrains 合作推出课程，解决 vibe coding 的最大痛点 — 不可控。先写 spec，再让 agent 执行。

3. **简单工作流 > 复杂编排**：Peter Yang 的实践证明，Plan Mode、MCP、子 agent 都是过度工程化。"直接跟它说话就行"。

4. **OpenAI 自动化研究时间表**：2026年9月实现自动化 AI 研究实习生，2028年3月实现真正的自动化 AI 研究员。

5. **AI 安全共识形成**：Anthropic 和 OpenAI 在关键安全问题上立场一致。Ilya Sutskever 确认这是"极其好的事"。

6. **AI Agent 商业化案例**：Nate Liason 的 OpenClaw 机器人 3 周赚 $14,718，三层记忆系统是关键。
